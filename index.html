    Welcome to the page for A Game.

    This is a project that was recently started as a break from Unity and C#. It is a game that is made with C++ and the SFML API.

    There is no finalized plan as to what the game will end up playing like, but there is an idea on how it should end up.

    Currently not in a playable state and is still being developed.

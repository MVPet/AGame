    Welcome to the page for Fighting Game.

    This is a project that was recently started as a break from Unity and C#. It is a game that is made with C++ and the SFML API.

    The game has gone through a few different concepts, such as a Beat 'em Up, but it is settled that it will be a fighting game to test out some concepts which will then be used in a Beat 'em Up later.

    Current plans are for just one fighter to test out concepts, and gameplay mechanics are not quite set in stone, but there is a plan for them.

    Currently not in a playable state and is still being developed.
